How this works

The hardest part is knowing what you want to say, the rest is easy.

Find what to say

If you are quite the Hemingway then you shouldn't have problems finding cool things to say. If you're not much of wordsmith google this exact phrase "Quotes by cool people that ill make me sound smart and lovable". This should do it.

Set up your image

Typographics allows you to set the width and height of the image you would like to use. You want to use this for Facebook cover image and don't know the darn measurements? No, problem, just use the template from the drop down menu.

Make it look cool

Typographics comes with cool fonts and let's you change the colors and size. This should be all you need. You want blinking text on fire? Sorry we've left that back in 1998. :(
